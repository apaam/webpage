Detailed Description

A class representing a poly superellipsoid with two different orders and three different axes.

This class provides methods for setting and updating the position of the poly superellipsoid, computing its surface properties, generating STL models for visualization, and calculating signed distances and surface points.

Additionally, the PolySuperEllipsoid class is a subclass of the Shape class, which allows it to be used in combination with other shapes in compound objects.

The poly superellipsoid is defined by the following equation: ( ( x / axis_a )^( 2 / order_ab ) + ( y / axis_b )^( 2 / order_ab ) ) ^ ( order_ab / oder_c ) + ( z / axis_c )^( 2 / order_c ) = 1 m and p indicate minus and plus to guarantee convexity, order should be in (0, 2)
