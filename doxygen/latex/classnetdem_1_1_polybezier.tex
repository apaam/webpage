\doxysubsection{Detailed Description}
A class representing a polybezier with variable orders and patches. 

This class provides methods for initializing, updating, and generating the \doxylink{classnetdem_1_1_polybezier}{Polybezier} object using various sources such as kernel STL models or random initialization. It also implements operations such as computing support points and surface properties. The polybezier is defined by its patches which are represented by knot vectors as well as edge patches and face patches. \begin{Desc}
