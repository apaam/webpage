\doxysubsection{Detailed Description}
A modifier class for performing breakage analysis on particles. 

The \doxylink{classnetdem_1_1_breakage_analysis_p_d}{Breakage\+Analysis\+PD} class is a modifier class that performs breakage analysis on the specified particles. It uses the \doxylink{classnetdem_1_1_peri_digm_d_e_m_coupler}{Peri\+Digm\+DEMCoupler} to interface with a DEM simulation, and has options for selecting particles based on ID or from a scene file.
