\doxysubsection{Detailed Description}
A class representing a poly superquadric with three different axes and orders. 

This class provides methods for setting and updating the position of the \doxylink{classnetdem_1_1_poly_super_quadrics}{Poly\+Super\+Quadrics} object, computing its surface properties, generating STL models for visualization, and calculating signed distances and surface points.

Additionally, the \doxylink{classnetdem_1_1_poly_super_quadrics}{Poly\+Super\+Quadrics} class is a subclass of the \doxylink{classnetdem_1_1_shape}{Shape} class, which allows it to be used in combination with other shapes in compound objects.

The poly superquadric is defined by the following equation\+:

( x / axis\+\_\+a )\texorpdfstring{$^\wedge$}{\string^}( 2 / order\+\_\+a ) + ( y / axis\+\_\+b )\texorpdfstring{$^\wedge$}{\string^}( 2 / order\+\_\+b ) + ( z / axis\+\_\+c )\texorpdfstring{$^\wedge$}{\string^}( 2 / order\+\_\+c ) = 1

where m and p indicate minus and plus to guarantee convexity. The order should be in (0, 2). \begin{Desc}
