\doxysubsection{Detailed Description}
A class representing a triangle in 3D space. 

This class provides methods for generating, initializing, and computing operations on a \doxylink{classnetdem_1_1_triangle}{Triangle} object. The object is defined by its three vertices, and provides methods for computing signed distance, surface point, and generating an STL model. It also provides various options for calculating the support points of the triangle. \begin{Desc}
