\doxysubsection{Detailed Description}
A class representing a damage model. 

The \doxylink{classnetdem_1_1_peri_digm_damage_model}{Peri\+Digm\+Damage\+Model} class represents a damage model that is used to simulate the deformation and failure of material in a peridynamic simulation. Currently, only one type of damage model is supported\+: critical stretch.
