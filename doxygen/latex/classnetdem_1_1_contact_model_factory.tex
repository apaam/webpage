\doxysubsection{Detailed Description}
A factory class for creating contact models. 

This class provides a set of static methods for creating new instances of contact models, based on a specified name and configuration data in JSON format. The mapping between model names and their corresponding {\ttfamily \doxylink{classnetdem_1_1_contact_model_a86b4462f72dd85e2e8a55dd6844f6b51}{Contact\+Model\+::\+Type}} enumeration values is stored in a static unordered map.
