\doxysubsection{Detailed Description}
A class for generating packs of particles. 

generate a grid of particles. Inputs\+:
\begin{DoxyItemize}
\item len\+\_\+x, len\+\_\+y, len\+\_\+z\+: lenghts of the box in x, y, and z directions.
\item center\+\_\+x, center\+\_\+y, center\+\_\+z\+: centroids of the box.
\item num\+\_\+x, num\+\_\+y, num\+\_\+z\+: number of points in each direction.
\item shape\+\_\+list\+: reference to shape library. Each generated particle would have a shape that is randomly selected from the shape\+\_\+list. Outputs\+: list of generated particles. 
\end{DoxyItemize}
