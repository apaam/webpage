\doxysubsection{Detailed Description}
A class used to solve collisions between a particle and a wall using a signed distance field. 

The \doxylink{classnetdem_1_1_solver_s_d_f_p_w}{Solver\+SDFPW} class is used to solve collisions between a particle and a wall based on the signed distance field of the particles and walls in a particle-\/wall collision detection system. \begin{Desc}
