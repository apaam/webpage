\doxysubsection{Detailed Description}
A simplex class for representing a convex hull in n-\/dimensional space. 

This class provides a set of constructors and methods for working with a simplex, which is a geometric object consisting of the convex hull of a set of n+1 points in n-\/dimensional space. The class is generic and can be used for any dimensionality n, although in practice it is most often used for 2D and 3D graphics applications. The simplex is represented as an array of Vec3d objects, one for each vertex, and a size variable indicating the number of vertices. \begin{Desc}
