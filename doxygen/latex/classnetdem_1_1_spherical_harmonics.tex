\doxysubsection{Detailed Description}
A class representing a spherical harmonics object. 

This class provides methods for generating, initializing, and computing operations on a \doxylink{classnetdem_1_1_spherical_harmonics}{Spherical\+Harmonics} object. The object is defined by its degree, radius, and coefficients for each term in the harmonics expansion. In particular, it provides methods for computing signed distance, surface point, and generating an STL model. It also provides various options for calculating the spherical harmonics terms, including basic, fast, and with pre-\/calculated values. \begin{Desc}
