\doxysubsection{Detailed Description}
A base class for commands that can modify a simulation based on provided JSON information. 

an interface that all commands would inheritage. A command modifies a simulation according to the provided json information. 

The {\ttfamily \doxylink{classnetdem_1_1_command}{Command}} class is a base class for commands that can modify a simulation based on provided JSON information. It provides functionality to store information about the command and a pointer to the simulation object, as well as a virtual function to execute the command.
