\doxysubsection{Detailed Description}
A class representing the material properties used in a peridynamic simulation. 

The \doxylink{classnetdem_1_1_peri_digm_material}{Peri\+Digm\+Material} class represents the material properties used in a peridynamic simulation. It includes methods for writing the material properties to file in a format compatible with the Peridigm code.
