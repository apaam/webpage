\doxysubsection{Detailed Description}
A class used to evaluate the stress of particles in a DEM simulation. 

The \doxylink{classnetdem_1_1_particle_stress_evaluator}{Particle\+Stress\+Evaluator} class calculates the stress of a specified set of particles, and stores this information in a list of Mat3d objects.
