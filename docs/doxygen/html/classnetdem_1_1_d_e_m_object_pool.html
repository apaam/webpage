Detailed Description

A class implementing object pooling for particles and contacts.

Particles and contacts are frequently added to or removed from the scene. The pool strategy is used to avoid the frequent construction and de-construction of object instances. When a particle or wall needs to be added, an instance will be obtained from the pool. When a particle or wall needs to be removed, it is recycled and stored in the pool for later re-use.
