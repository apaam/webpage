\doxysubsection{Detailed Description}
Class for managing the simulation domain and particles that belong to it. 

The {\ttfamily \doxylink{classnetdem_1_1_domain}{Domain}} class includes a {\ttfamily \doxylink{classnetdem_1_1_cell_manager}{Cell\+Manager}} object for broad-\/phase contact detection. It stores information about the rank id and total number of processes, the lower and upper bounds of the domain, and a list of particles that are outside of the domain but need to be transferred to it for contact detection.
