\doxysubsection{Detailed Description}
A class that represents a tetrahedral mesh. 

The {\ttfamily \doxylink{classnetdem_1_1_tet_mesh}{Tet\+Mesh}} class is responsible for creating and managing a tetrahedral mesh. It provides functionality to initialize the mesh, compute connectivity between nodes and elements, create a surface mesh, and save the mesh as a VTK file. \begin{Desc}
