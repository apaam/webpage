\doxysubsection{Detailed Description}
A container class for settings related to contact solvers. 

This class provides a set of member variables that can be used to configure various aspects of the contact solvers used in a simulation. The {\ttfamily Solver\+Type} enumeration specifies the type of solver to use, while other variables control specific options like erosion and logging.
