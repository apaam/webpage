\doxysubsection{Detailed Description}
A command class to create an object in the simulation. 

With the input parameters defined in json, create and insert a sphere shape into a simulation. 

The {\ttfamily \doxylink{classnetdem_1_1_command_create}{Command\+Create}} class is a command class that creates an object in the simulation based on the provided JSON information. It inherits from the {\ttfamily \doxylink{classnetdem_1_1_command}{Command}} base class and implements the {\ttfamily \doxylink{classnetdem_1_1_command_create_af122f0fff5e5834dc0a8fbaea9419bb5}{Execute()}} function to create the object in the simulation.
