\doxysubsection{Detailed Description}
A class representing a poly superellipsoid with two different orders and three different axes. 

This class provides methods for setting and updating the position of the poly superellipsoid, computing its surface properties, generating STL models for visualization, and calculating signed distances and surface points.

Additionally, the \doxylink{classnetdem_1_1_poly_super_ellipsoid}{Poly\+Super\+Ellipsoid} class is a subclass of the \doxylink{classnetdem_1_1_shape}{Shape} class, which allows it to be used in combination with other shapes in compound objects.

The poly superellipsoid is defined by the following equation\+: ( ( x / axis\+\_\+a )\texorpdfstring{$^\wedge$}{\string^}( 2 / order\+\_\+ab ) + ( y / axis\+\_\+b )\texorpdfstring{$^\wedge$}{\string^}( 2 / order\+\_\+ab ) ) \texorpdfstring{$^\wedge$}{\string^} ( order\+\_\+ab / oder\+\_\+c ) + ( z / axis\+\_\+c )\texorpdfstring{$^\wedge$}{\string^}( 2 / order\+\_\+c ) = 1 m and p indicate minus and plus to guarantee convexity, order should be in (0, 2) \begin{Desc}
