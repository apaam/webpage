\doxysubsection{Detailed Description}
A class representing the settings used in a peridynamic simulation. 

The \doxylink{classnetdem_1_1_peri_digm_settings}{Peri\+Digm\+Settings} class represents the settings used in a peridynamic simulation. It includes methods for writing the settings to file in a format compatible with the Peridigm code.
