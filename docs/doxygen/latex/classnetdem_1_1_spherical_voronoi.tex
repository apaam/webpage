\doxysubsection{Detailed Description}
Class for computing \doxylink{classnetdem_1_1_voronoi}{Voronoi} diagrams on a sphere. 

This class provides methods for computing \doxylink{classnetdem_1_1_voronoi}{Voronoi} diagrams on the surface of a sphere. It can solve basic, weighted, and centroidal \doxylink{classnetdem_1_1_voronoi}{Voronoi} problems, and provides methods for saving the resulting \doxylink{classnetdem_1_1_voronoi}{Voronoi} mesh as a VTK file for visualization.
