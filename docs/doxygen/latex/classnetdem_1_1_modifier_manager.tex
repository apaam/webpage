\doxysubsection{Detailed Description}
A class used to manage modifiers in a simulation. 

The \mbox{\hyperlink{classnetdem_1_1_modifier_manager}{Modifier\+Manager}} class handles the activation and deactivation of modifiers, as well as the updating of modifier properties when the scene changes.
