\doxysubsection{Detailed Description}
An abstract base class for contact models. 

This class defines a set of virtual methods and member variables that must be implemented by any subclass representing a specific contact model. The {\ttfamily Type} enumeration provides a list of possible model types, while the {\ttfamily id}, {\ttfamily label}, and {\ttfamily model\+\_\+name} variables can be used to identify and describe individual instances of contact models.
