\doxysubsection{Detailed Description}
A class representing a simulator for peridynamic simulations. 

The \doxylink{classnetdem_1_1_peri_digm_simulator}{Peri\+Digm\+Simulator} class represents a simulator for peridynamic simulations. It includes methods for setting up and running the simulation, as well as writing input and output files in a format compatible with the Peridigm code.
