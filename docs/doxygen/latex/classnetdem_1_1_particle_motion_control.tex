\doxysubsection{Detailed Description}
A class used to add motion control to particles in a DEM simulation. 

The \doxylink{classnetdem_1_1_particle_motion_control}{Particle\+Motion\+Control} class is used to add various types of motion to particles, including fixed, linear velocity, and sinusoidal velocity.
