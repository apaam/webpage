\doxysubsection{Detailed Description}
A profiler class for measuring performance metrics in a DEM simulation. 

This class provides a set of methods and member variables for measuring various performance metrics during a discrete element method (DEM) simulation. These metrics include timing information for different parts of the simulation, as well as counts of particles, walls, neighbors, and other relevant quantities. The class also includes methods for clearing and printing the collected data. \begin{Desc}
