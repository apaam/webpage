\doxysubsection{Detailed Description}
A class representing a domain splitter for tetrahedral meshes. 

The \doxylink{classnetdem_1_1_tet_mesh_splittor}{Tet\+Mesh\+Splittor} class represents a domain splitter for tetrahedral meshes. It derives from the \doxylink{classnetdem_1_1_domain_splittor}{Domain\+Splittor} abstract base class and provides implementation for the methods required by that class.
