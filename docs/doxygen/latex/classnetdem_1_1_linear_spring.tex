\doxysubsection{Detailed Description}
Contact model that uses linear spring elements to evaluate contact forces and moments. 

The linear spring contact model calculates the contact force based on the contact deformation, which is proportional to the normal displacement between two particles. The tangential force is calculated based on the Coulomb\textquotesingle{}s friction law. \begin{Desc}
