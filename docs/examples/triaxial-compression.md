## Triaxial compression

Related guides: [Basic usage](../manual/basic-usage.md), [Python API overview](../python-api/index.md).

Triaxial compression is a standard geotechnical laboratory test used to characterize the mechanical behavior of granular materials. This example simulates a conventional triaxial compression test on an assembly of irregular-shaped particles, including specimen preparation, confining pressure application, and axial loading.
