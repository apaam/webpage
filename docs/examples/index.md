## Examples

All examples are provided as Python scripts using the `phynexis` API. See [Basic usage](../manual/basic-usage.md) for a minimal simulation pattern and the [Python API overview](../python-api/index.md) for module layout.

### General examples

- [Random packing](random-packing.md): random packing of irregular-shaped particles using a rain-falling approach.
- [Blade mixing](blade-mixing.md): spherical particles in a rotating blade mixer with STL-imported geometry.
- [Granular mixing](granular-mixing.md): mixing of granular materials with different properties.

### Geotechnical experiments

- [Triaxial compression](triaxial-compression.md): conventional triaxial compression test on irregular-shaped particles.
- [Angle of repose](angle-of-repose.md): formation and measurement of a granular pile slope.
