应力 (Stress)​

基于连续性介质理论，材料的动量守恒方程可表示为（忽略体力及加速度项）

∇⋅σ=0\begin{align} \boldsymbol{\nabla} \cdot \boldsymbol{\sigma} = \boldsymbol{0} \end{align}∇⋅σ=0​​

采用爱因斯坦求和约定，上式可写作

∇iσij=0j\begin{align} \nabla_i \sigma_{ij} = 0_j \end{align}∇i​σij​=0j​​​

利用动量守恒方程可得等式

∇i(σijxk)=∇iσijxk+σij∇ixk=∇iσijxk+σijδik=σjk\begin{align} \nabla_i (\sigma_{ij} x_k) = \nabla_i \sigma_{ij} x_k + \sigma_{ij} \nabla_i x_k = \nabla_i \sigma_{ij} x_k + \sigma_{ij} \delta_{ik} = \sigma_{jk} \end{align}∇i​(σij​xk​)=∇i​σij​xk​+σij​∇i​xk​=∇i​σij​xk​+σij​δik​=σjk​​​

针对某个颗粒 ppp，其内部应力的积分可计算为

∫σjk dΩ=∫∇i(σijxk) dΩ=∫ni(σijxk) dΓ=∫ni(σijxk) dΓ=∫fjxk dΓ=∑cfjxk\begin{align} \int \sigma_{jk} ~\text{d} \Omega = \int \nabla_i (\sigma_{ij} x_k) ~\text{d} \Omega = \int n_i (\sigma_{ij} x_k) ~\text{d} \Gamma = \int n_i (\sigma_{ij} x_k) ~\text{d} \Gamma = \int f_j x_k ~\text{d} \Gamma = \sum^c f_j x_k \end{align}∫σjk​ dΩ=∫∇i​(σij​xk​) dΩ=∫ni​(σij​xk​) dΓ=∫ni​(σij​xk​) dΓ=∫fj​xk​ dΓ=∑c​fj​xk​​​

其中，ccc 表示该颗粒的所有接触、fjf_jfj​ 表示接触力、 xkx_kxk​ 表示接触位置。进而，利用颗粒在接触力作用下的静力平衡公式

∑cfjxkp=(∑cfj)xkp=0\begin{align} \sum^c f_j x_k^p = (\sum^c f_j) x_k^p = 0 \end{align}∑c​fj​xkp​=(∑c​fj​)xkp​=0​​

可得

∫σjk dΩ=∑cfjxk=∑cfj(xk−xkp)=∑cfjbk\begin{align} \int \sigma_{jk} ~\text{d} \Omega = \sum^c f_j x_k = \sum^c f_j (x_k - x_k^p) = \sum^c f_j b_k \end{align}∫σjk​ dΩ=∑c​fj​xk​=∑c​fj​(xk​−xkp​)=∑c​fj​bk​​​

其中，xkpx_k^pxkp​ 为颗粒质心、bkb_kbk​ 为由接触点指向颗粒质心的向量（branch vector）。

针对代表单元体，其域内的平均应力可由其内部所有颗粒内部的应力积分，再除于代表单元体体积得到，表达为

σˉjk=1V∫σjk dΩ=∑c(fjbkp−fjbkq)=∑cfjdk\begin{align} \bar{\sigma}_{jk} = \frac{1}{V} \int \sigma_{jk} ~\text{d} \Omega = \sum^c (f_j b_k^p - f_j b_k^q) = \sum^c f_j d_k \end{align}σˉjk​=V1​∫σjk​ dΩ=∑c​(fj​bkp​−fj​bkq​)=∑c​fj​dk​​​

其中，ccc 为代表单元体内的所有接触、dkd_kdk​ 表示颗粒质心相对位置矢量。

应变 (Strain)​

假设单元变形梯度（deformation gradient）为 F\boldsymbol{F}F，则变形后，颗粒 ppp（初始位置 Xp\boldsymbol{X}^pXp）与 颗粒 qqq（初始位置 (Xq(\boldsymbol{X}^q(Xq）质心相对位置矢量 d\boldsymbol{d}d 可计算为

di=Fij(Xjp−Xjq)\begin{align} d_i = F_{ij} (X^p_j - X^q_j) \end{align}di​=Fij​(Xjp​−Xjq​)​​

本构张量 (Constitutive tangent moduli)​

dfi=kn(dFmn)dnnnni+kt(dFmn)dntnti\begin{align} \text{d} f_i = k_n (\text{d} F_{mn}) d_n n_n n_i + k_t (\text{d} F_{mn}) d_n t_n t_i \end{align}dfi​=kn​(dFmn​)dn​nn​ni​+kt​(dFmn​)dn​tn​ti​​​ Cijkl=∂σij∂Fkl=∑cknnidjnkdl+kttidjtkdl\begin{align} C_{ijkl} = \frac{\partial \sigma_{ij}}{\partial F_{kl}} = \sum^c k_n n_i d_j n_k d_l + k_t t_i d_j t_k d_l \end{align}Cijkl​=∂Fkl​∂σij​​=∑c​kn​ni​dj​nk​dl​+kt​ti​dj​tk​dl​​​
