## Project Overview

Phynexis documentation site built with **Docusaurus v3**. Published at `https://apaam.github.io/webpage/`.

- **Language**: English (site UI + most docs). Two theory pages (`manual/discrete-to-continuum.md`, `manual/particle-models.md`) are Chinese drafts; sidebar labels them as drafts.
- **Math**: KaTeX via `remark-math` + `rehype-katex`. Use `$inline$` and `$$display$$`.
- **Citations**: `rehype-citation` with bibliography at `static/files/refs.bib`.
- **Static assets**: Place under `static/` (e.g. `static/img/foo.png` → `/img/foo.png` in Markdown).

## Architecture

### Doc Sections & Sidebars

Docs are organized into 5 independent sidebars defined in `sidebars.ts`:

1. **`userGuideSidebar`** — `docs/manual/` (getting started, advanced, developer, theory)
2. **`pythonApiSidebar`** — `docs/python-api/` (autogenerated from directory tree)
3. **`examplesSidebar`** — `docs/examples/` (simulation examples)
4. **`gallerySidebar`** — `docs/gallery/` (animations, snapshots)
5. **`downloadSidebar`** — `docs/download/` (autogenerated)
6. **`aboutSidebar`** — `docs/about/` (single page)

The navbar links route to the correct sidebar via `type: 'doc'` with explicit `sidebarId`.

### Python API Docs

- Source pages: `docs/python-api/{module}/` — one Markdown file per class
- Meta tracking: `logs/python-api-meta/progress.json` (module/class completion status)
- Old reference: `docs/python-api/deprecated/ref-*.md` (not authoritative)
- PDF export script: `scripts/export-python-api-pdf.py` — collects all `.md`, merges, converts via `pandoc → typst` using the `manual` template

### C++ API Reference

- Generated by **Doxygen** via `Doxyfile`
- Output: `static/doxygen/html/...` (`.gitignore`d)
- Navbar entry **API → C++ (Doxygen)** opens in a new tab because Doxygen uses separate styling
- `Doxyfile` `INPUT` paths expect a Phynexis source tree (e.g. `../phynexis/src`)

### PDF Export

Two scripts in `scripts/`:
- `export-python-api-pdf.py` — Python API docs → `static/pdf/phynexis-python-api.pdf`
- `export-docs-pdf.py` — All docs (excl. python-api) → `static/pdf/phynexis-docs.pdf`

Both use `pandoc → typst` with templates from `~/workspace/templates/md-to-pdf/`. The scripts handle:
- Docusaurus absolute image paths (`/img/...`)
- Disabling pandoc citation processing (avoids Typst `@label` errors)
- `typst compile --root /` for absolute image path resolution

### Landing Page

- `src/pages/index.tsx` — custom React landing page
- `src/pages/index.module.css` — landing page styles
- `src/css/custom.css` — global Docusaurus style overrides

### Key Config Files

- `docusaurus.config.ts` — Docusaurus config. `baseUrl: '/webpage/'`, `future.v4: true`, `trailingSlash: true`
- `sidebars.ts` — Sidebar definitions
- `tsconfig.json` — TypeScript config (extends `@docusaurus/tsconfig`)

## File Conventions

- **Files / directories**: kebab-case (e.g. `dem-basics.md`, `blade-rendering.md`)
- **New docs**: Write in English. Add to the appropriate sidebar in `sidebars.ts`
